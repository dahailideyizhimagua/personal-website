参数化设计工具

基于节点编辑器的参数化建筑设计工具，支持可视化编程。

角色

全栈开发

年份

2024

技术栈

Vue 3Canvas APINode.jsWebAssembly

项目概述

受 Grasshopper 启发的 Web 版参数化设计工具，通过节点连线定义几何生成逻辑。

核心特性

  • 可视化节点编辑器（拖拽连线）
  • 实时几何预览
  • 内置数学/几何节点库
  • 参数滑块实时调整
  • 导出 SVG/DXF 格式
